# Screenshots Of The Demo / صور العرض

## English

The `screenshots/` folder contains generated screenshots of the current demo UI.

Files:

- `desktop-demo.png` - laptop/desktop demo view.
- `portrait-kiosk-demo.png` - portrait kiosk-style view.

Screenshots should not include browser chrome, API keys, debug panels, or environment values.

## العربية

يحتوي مجلد `screenshots/` على صور واجهة العرض الحالية.

الملفات:

- `desktop-demo.png` - عرض مناسب للابتوب أو سطح المكتب.
- `portrait-kiosk-demo.png` - عرض رأسي مناسب لشاشة كشك.

يجب ألا تحتوي الصور على شريط المتصفح أو مفاتيح API أو لوحات تصحيح أو قيم بيئة التشغيل.

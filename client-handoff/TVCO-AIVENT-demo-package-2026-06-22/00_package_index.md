## Sending Notes / ملاحظات الإرسال

- Do not include `.env.local` or API keys.
- لا ترسل ملف `.env.local` أو أي مفاتيح API.
- For a stable client demo, set `USE_MOCK_AI=true`.
- للحصول على عرض مستقر أمام العميل، استخدم `USE_MOCK_AI=true`.
- Replace mock content with approved TVCO/client content before any production pilot.
- يجب استبدال المحتوى التجريبي بمحتوى معتمد من TVCO أو العميل قبل أي تجربة فعلية.
